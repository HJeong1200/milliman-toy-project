## 1. 패키지 실행 방법
### 1-1. Windows
- 패키지 설치
```bash
npm install
```

- 프로그램 실행
```bash
npm run start
```

- electron 실행
```bash
npm run windows-start
```

### 1-2. Mac OS
- 패키지 설치
```bash
npm install
```

- 프로그램 실행
```bash
npm run react-start
```

- electron 실행
```bash
npm run electron-start
```
